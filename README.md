# Python Daily Projects

This repository contains a collection of beginner-level Python projects created while learning and practicing Python programming. The goal of this repository is to improve problem-solving skills and strengthen understanding of Python fundamentals through small console-based applications.

Each project focuses on practicing core programming concepts such as user input handling, loops, conditional statements, lists, functions, and randomization.

More projects will be added regularly as part of continuous learning and daily coding practice.

## Features

- Collection of beginner-friendly Python mini projects
- Practice with real coding problems
- Helps improve logical thinking and problem solving
- Demonstrates Python fundamentals in simple applications
- Projects will continue to grow as learning progresses

## Technologies Used

- Random module
- Conditional logic (if / elif / else)
- Loops
- Lists
- User input handling
