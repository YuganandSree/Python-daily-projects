# Band-Name-Generator-Project
The Band Name Generator is a beginner-level Python console application designed to demostrate fundamental programming concepts such as user input handling, variable assignment, and string concatenation.
The program greets the user, prompts them to enter the name of the city they grew up in and their pet's name, and then combines these inputs to generate a creative band name suggestion.
This project reinforces core Python concepts including:
1. print() statements for output
2. input() function for user interaction
3. Variable storage
4. String concatination
5. Basic program flow
It serves as an introductory exercise to build confidence in writing interactive Python programs and understanding how user data can be processed to produce dynamic outputs.
